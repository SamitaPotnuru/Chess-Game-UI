# Chess-Game-UI
A fully functional browser-based Chess game built using HTML, CSS, and JavaScript, featuring both Human vs Human and Human vs AI gameplay.

## 🚀 Features
- ♟ Play vs Computer (AI)
- 👥 2-Player Mode (Same Screen)
- 🎯 Three Difficulty Levels  
  - Easy (Depth 2)  
  - Medium (Depth 3)  
  - Hard (Depth 4)  
- ♜ Complete Chess Rules  
  - Castling  
  - En Passant  
  - Pawn Promotion  
  - Check & Checkmate detection  
  - Stalemate detection  
- 📜 Move History Tracker  
- ♟ Captured Pieces Display  
- 💡 Move Highlighting  
- 🤖 AI with Minimax + Alpha-Beta Pruning  
- 🎨 Elegant Classic UI Design  

## 🛠️ Technologies Used

- HTML5 – Structure  
- CSS3 – Styling & UI  
- Vanilla JavaScript – Game logic & AI  

## 🧠 AI Logic

The AI uses:

- Minimax Algorithm  
- Alpha-Beta Pruning (Optimization)  
- Piece Value Evaluation  
- Piece-Square Tables (Positional Strength)
